1.3 计算机的性能指标

  • 1.机器字长
    • 计算机进行一次整数运算（即正整数运算）所能处理的二进制数据的位数。
  • 2.数据通路带宽
    • 外部数据一次所能并行传递信息的位数。
  • 3.主存容量
    • 主存所能存储信息的最大容量，通常以字节来衡量，也可用字数x字长来表示。其中MAR的位数反映了存储单元的个数，反映了可寻址范围的最大值。
    • 如：MAR为16位，表示216=65536，即此存储体内有65536个存储单元。若MDR为32位，表示存储容量为64Kx32位。
  • 4.运算速度
    • (1) 吞吐量和响应时间
      • 吞吐量：系统在单位时间内处理请求的数量。
      • 响应时间：从用户向计算机发送请求，到系统对该请求做出响应并获得它所需的结果的等待时间。
    • (2) 主频和CPU时钟周期
      • CPU时钟周期：通常指节拍脉冲或T周期，即主频的倒数。
      • 主频：机器内部主时钟的频率，是衡量机器速度的重要参数（GHz，MHz）。
    • (3) CPI（Clock cyber Per Instruction）：执行一条指令所需的时钟周期数。
    • (4) CPU执行时间：执行一个程序所花费的时间。
      • CPU执行时间 = CPU时钟周期数 / 主频 = （指令条数 x CPI） / 主频
    • (5) MIPS(Million Instructions Per Second)：每秒执行几百万条指令
      • MIPS = 指令条数 / （执行时间 x 106） = 主频 / CPI
    • (6) MFLOPS(Million Floating-point Per Second)：每秒执行几百万次浮点运算
      • MFLOPS = 浮点操作次数 / （执行时间 x 106）
    • (7) GFLOPS(Giga Floating-point Per Second)：每秒执行几十亿次浮点运算
      • GFLOPS = 浮点操作次数 / （执行时间 x 109）
    • (7) TFLOPS(Tiga Floating-point Per Second)：每秒执行几万亿次浮点运算
      • TFLOPS = 浮点操作次数 / （执行时间 x 1012）

    • A. 时钟频率
    • B. 系统结构

一、单选

10.在CPU的寄存器中，（ ）对用户是完全透明的。

A.程序计数器B.指令寄存器C.状态寄存器D.通用寄存器A.\text{程序计数器}\quad B.\text{指令寄存器}\quad C.\text{状态寄存器}\quad D.\text{通用寄存器} A.程序计数器B.指令寄存器C.状态寄存器D.通用寄存器

[解]
A.汇编程序员可以通过指定待执行指令的地址来指定PC的值
B.指令寄存器（IR）、MDR、MAR是CPU内部工作寄存器，对程序员均不可见
C.D. 状态寄存器和通用寄存器只对汇编程序员可见，才能编程
∴B is true\therefore \text{B is true}∴B is true

12. CPU的CPI与下列那个因素无关？（ ）

A.时钟频率B.系统结构C.指令集D.计算机组织A.\text{时钟频率}\quad B.\text{系统结构}\quad C.\text{指令集}\quad D.\text{计算机组织} A.时钟频率B.系统结构C.指令集D.计算机组织

[解]
CPI是指执行一条语句所耗费的时钟周期。时钟周期是定值，加快或减慢时钟频率只能加快或减慢执行速度，但是并不能改变定值——时钟周期。时钟周期的取值由系统结构、指令集和计算机组织决定。
∴A is true\therefore \text{A is true}∴A is true

14. 当前设计高性能计算机的重要技术途径是（ ）

A.提高CPU主频B.扩大主存容量C.采用非冯诺依曼结构D.采用并行处理技术A.\text{提高CPU主频}\quad B.\text{扩大主存容量}\quad C.\text{采用非冯诺依曼结构}\quad D.\text{采用并行处理技术} A.提高CPU主频B.扩大主存容量C.采用非冯诺依曼结构D.采用并行处理技术

[解]
提高CPU主频、扩大主存容量对性能的提升都是有限的。采用并行技术是实现高性能计算机的重要途径，现代超级计算机均采用多处理器来增强并行处理能力。
∴D is true\therefore \text{D is true}∴D is true
