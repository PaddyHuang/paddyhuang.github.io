1.2 算法和算法评价

1.2.1 算法的基本概念

算法是对特定问题求解步骤的一种描述，具有以下5个重要特性：
1. 有穷性
2. 确定性
3. 可行性
4. 输入
5. 输出

设计一个好的算法，考虑以下目标：
1. 正确性
2. 可读性
3. 健壮性
4. 效率与低存储量需求

1.2.2 算法效率的度量

  3. 平均时间复杂度——所有可能输入实例在等概率出现的情况下，算法的期望运行时间。
  4. 最好时间复杂度——最好情况下，算法的时间复杂度。
