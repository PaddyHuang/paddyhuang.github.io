    第1章-程序设计和C语言

    所谓程序，就是一组计算机能识别和执行的指令。

    1.2 什么是计算机语言

    计算机语言有以下几个发展阶段：

    1. 机器语言——由计算机能够直接识别和接受的二进制机器指令组成的集合。
    2. 汇编语言。
    3. 高级语言——接近于人类自然语言和数学语言，不依赖于具体机器。高级语言编写的程序称为源程序；编译程序把源程序转换为机器指令程序（目标程序）。
      高级语言经历了如下的发展阶段：
      • 非结构化语言：只要符合语法规则，可以允许随意跳转；
      • 结构化语言：规定程序必须由具有良好特性的基本结构（顺序结构、选择结构、循环结构）构成，不允许随意跳转。
      • 面向对象语言。

    1.3 C语言的发展及其特点

    1. 语言简洁、紧凑，使用方便、灵活；
    2. 运算符（34种）丰富；
    3. 数据类型丰富；
    4. 具有结构化控制语句；
    5. 语法限制不太严格，程序设计自由度大；
    6. 允许直接访问物理地址，能进行位操作；
    7. 可移植性好；
    8. 生成的目标代码质量高，程序执行效率高。

    1.4.2 C语言程序结构

    1. 一个程序由一个或多个源程序文件组成。
      一个源程序文件可以包括3个部分：
      a) 预处理命令（#include <stdio.h> 或#define false 0）；
      b) 全局声明；
      c) 函数定义。
    2. 函数是C程序的主要组成部分。
    3. 一个函数包括两个部分：
      a) 函数首部；
      b) 函数体（声明部分 + 执行部分）。
    4. 程序总是从main函数开始执行。
    5. 程序中要求计算机完成的操作是由函数中的C语句完成的。
    6. 在每个声明和语句的最后必须由一个分号。
    7. C语言本身不提供输入输出语句。
    8. 程序应当包含注释。

    计算机语言是为了解决人和计算机交流信息的问题而产生的。高级语言有如下特点：

    习题3. 正确理解以下名词及其含义：

    1. 源程序，目标程序，可执行程序；
      源程序：高级语言编写的程序；
      目标程序：由源程序转换为机器指令的程序；
      可执行程序：计算机能直接识别并执行的程序。

    2. 程序编辑，程序编译，程序连接；
      程序编辑：对程序进行增删改查；
      程序编译：将源程序转化为机器指令程序；
      程序连接：将目标程序转化为可执行程序。

    3. 程序，程序模块，程序文件；
      程序：一组计算机能够识别和执行的指令；
      程序模块：组成程序的各个模块；
      程序文件：存储程序的文件。

    4. 函数，主函数，被调用函数，库函数；
      函数：实现某种功能的执行模块；
      主函数：C程序的入口函数；
      被调用函数：被主调函数调用的函数；
